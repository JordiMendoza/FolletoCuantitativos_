Modelos de redes

Diseño estratégico para conectar nodos y canales logrando el menor costo, distancia o tiempo total.

Teoría

Modelado de Redes

Por medio del modelado de redes podemos tener una mejor forma de abarcar problemas relacionados con distancias, distribución logística e incluso la necesidad total de materiales. Por lo cual, en esta exposición se abarcaron tres herramientas esenciales para el desarrollo de problemas de redes.

Árbol de Expansión Mínima

Este método sirve para conectar todos los puntos de una red (nodos) de la manera más barata posible, sin crear ciclos. Es como tirar tuberías entre varias casas usando el mínimo material. Para lograrlo, el modelo analiza los pesos o costos de cada enlance (distancias, tiempos, precios) y selecciona la combinación óptima. Los compañeros en este caso utilizaron el algoritmo de Kruskal

Flujo Máximo

Se usa para saber cuánto producto o datos pueden pasar por una red desde el inicio hasta el fin. Es clave para identificar cuellos de botella y saber el límite real del sistema. Requiere definir un nodo de origen (fuente) y uno de destino (sumidero), donde cada enlace tiene una capacidad límite que no se puede exceder. Matemáticamente, los compañeros lo resolvieron utilizando el algoritmo de Ford-Fulkerson.

Ruta Más Corta

El objetivo es encontrar el camino con el menor costo o distancia entre dos puntos. Es la lógica detrás de aplicaciones como Waze para llegar lo más rápido posible. A diferencia del árbol de expansión, no busca conectar toda la red, sino optimizar la trayectoria lineal única entre dos nodos específicos. Los compañeros utilizaron el Algoritmo de Dijkstra para resolver el ejemplo en clase.

Conclusión

Conclusiones del tema

Indagando sobre estos temas, me quedó clarísima la importancia de algoritmos como Dijkstra en la programación actual. Además, es muy evidente la similitud que tienen con los modelos de transporte.
